#Hospital Management System

This project is a simple console-based Hospital Management System developed using the C
programming language. It helps manage patient records efficiently with features such as adding,
viewing, searching, updating, and deleting patient details. The system demonstrates the use of arrays,
structures, priority logic, and file handling.
##Features
- Add new patient records
- Display all patients
- Search patients by ID or name
- Update patient details
- Delete patient records
- Priority-based handling (optional)
- Stores data permanently using text files
- Simple and user-friendly console interface
